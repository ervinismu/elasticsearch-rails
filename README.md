# elasticsearch-rails
Elasticsearch + Kibana + Searchkick + Ruby on Rails

### Pre
*	Kibana (optional)
* Elasticsearch v 5.x or higher
*	Ruby 2.5 or higher

### Run
*	Elasticsearch | `.bin/elasticsearch`
*	Rails	|	`rails s`
* Kibana	|	`sudo systemctl start kibana.service`

notes: </br>
Open kibana page in `localhost:5601/app/kibana`
