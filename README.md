# elasticsearch-rails
example using elasticsearch in ruby on rails application

### Pre
*	Kibana (optional)
* Elasticsearch v 5.x or higher
*	Ruby 2.5 or higher

### Run
*	Elasticsearch | `.bin/elasticsearch`
*	Rails	|	`rails s`
* Kibana	|	`sudo systemctl start kibana.service`
